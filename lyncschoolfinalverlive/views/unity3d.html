UNITY 3D

Unity3D is a “game development ecosystem”, it includes an environment for the development of interactive 2D and 3D content including a rendering and physics engine, a scripting interface to program interactive content, a content exporter for many platforms (desktop, web, mobile, consoles, etc) and a growing knowledge sharing community. It’s very flexible and can be used for everything from simple 2D games to high end AAA style experiences.
